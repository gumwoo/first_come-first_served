## 목적
내 예매 내역 조회. 예정/사용완료/취소 탭 + 적립/쿠폰 등 프로필 요약.

## 상태(states)
- `default` (목록)
- `empty`

## 사용 API
- `GET /me` / `GET /me/orders?status=`

## 화면 요소 (DoD 체크리스트)
- [ ] 좌측 프로필 카드(이름/등급/적립금/쿠폰)
- [ ] 상단 요약 지표(예정/예매/취소/관심)
- [ ] 탭(전체/예정/사용완료/취소)
- [ ] 예매 목록(포스터/공연/일시/좌석/금액/상태배지)
- [ ] 우측 선택 예매 상세 + "모바일 티켓 보기"
- [ ] 페이지네이션

## 연결
- 예매 항목 → 상세 / 취소 → /me/orders/:id/refund
