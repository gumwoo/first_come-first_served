## 목적
전체 주문/예매 내역 검색·필터·관리.

## 사용 API
- `GET /admin/orders?status=&event=&method=&from=&to=&q=&page=`

## 화면 요소 (DoD 체크리스트)
- [ ] 필터 바(상태/이벤트/결제수단/기간/검색)
- [ ] 상단 집계 카드(전체/완료/대기/취소/환불 건수)
- [ ] 주문 테이블(주문번호/회원/공연/좌석/금액/결제수단/상태/일시)
- [ ] 우측 선택 주문 상세 패널
- [ ] 페이지네이션 / 내보내기

## 연결
- 행 클릭 → 주문 상세 패널
