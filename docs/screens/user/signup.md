# U-SIGNUP · 회원가입

- ref: `assets/screens/회원가입.png`
- route: `/signup`
- slice: S01

## 목적
신규 계정 생성. 약관 동의 + 휴대폰 인증 포함.

## 상태(states)
- `default`
- `phone-verifying` — 인증번호 발송/확인
- `error` — 유효성 실패(이메일 중복/비번 불일치)

## 사용 API
- `POST /auth/signup` (이름/이메일/비번/휴대폰/약관동의)
- 휴대폰 인증번호 발송·확인 API

## 화면 요소 (DoD 체크리스트)
- [ ] 이름 / 이메일 / 비밀번호 / 비밀번호 확인
- [ ] 휴대폰 번호 + 인증요청/인증확인
- [ ] 약관 동의 체크박스(필수/선택 구분)
- [ ] 회원가입 버튼(primary)
- [ ] 우측 가입 안내 박스
- [ ] 로그인 링크

## 연결
- 가입 성공 → /login 또는 자동 로그인 후 /
